# مولّد ريلز القرآن الكريم
## Quran Reels Generator

تطبيق ويب لتوليد فيديوهات قصيرة (Reels) لآيات القرآن الكريم مع تلاوة وخلفية وتشكيل كامل.

### المميزات
- 🎬 توليد فيديوهات 1080×1920 (Reels)
- 🎤 59 قارئ قرآن
- 📖 تفسير بـ 118 لغة
- 🎨 تخصيص الخطوط والألوان والخلفيات
- 🖼️ رفع صور/فيديو كخلفية مع نظام حماية إسلامي
- ✨ تشكيل قرآني كامل بـ HarfBuzz + FreeType
- 🔤 8 خطوط عربية

### التشغيل
```bash
pip install -r requirements.txt
python main.py
```
ثم افتح http://localhost:5001

### المتطلبات
- Python 3.8+
- FFmpeg (يُحمّل تلقائياً)
